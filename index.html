CMW

CMW is a project that we made for school olympics. The objective was creating a software in order to send alerts, talking in a medical way. And then we made it. Four shcoolmates created "Clinica Médica Wilde" in six days. In this platform we have 3 logins (Patient, Doctor and Admin). As a Patient we can book an appointment with an specific area and an specific doctor. Obviously, we can send alerts (green, yellow and red) depending on risk or damage. Also we can see appointments we reserved and our credential. Doctors can see alerts, accept them and easily see them as a card with the color of the alert and info inside. And see the appointments assigned as a doctor. Finally, an admin can see Doctors and Patients registered in the database, and the most importan section is "Métricas" (metrics) where we can see how many alerts where send and attended using graphics and tables. This was made using HTML, CSS, PHP and SQL.

JuegaYa is a platform we made thinking in kindergarden's children. This was created for children in order for them to learn letters, numbers and basic things in a funny way, and also for them to getting into computers and know how to use basic functionalities as using a mouse, making click, drag and hover while they are learning neccesary things. As this was created to argentine children the page is all in Spanish but we also created the section of games in English for them to get in touch with English in a early age.

This was made in HTML, CSS and pure Javascript using no libraries to be heavy in order to work in any computer.
